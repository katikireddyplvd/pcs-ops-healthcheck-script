# PCS Ops Health Check Script

## Overview
This project contains a shell script used to perform health checks for PCS platform services such as MQ, SFTP, and NATS.

## Features
- MQ connectivity test
- SFTP connectivity test
- NATS health validation
- Status summary output (SUCCESS / FAILED)

## Technologies Used
- Shell Scripting (Bash)
- OpenShift (oc commands)
- Kubernetes environment

## Purpose
This script helps DevOps engineers quickly validate system health during deployments and troubleshooting.
